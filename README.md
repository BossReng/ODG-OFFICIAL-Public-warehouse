## 📂 仓库说明 (About)

这是一个静态资源仓库，主要用于存放 **ODG-CRAFT** 服务器及 **BossReng** 个人项目所需的各类杂项文件。

这里的“乱七八糟”包括但不限于：
* 🖼️ **Images:** PCL2/HMCL 启动器背景图、公告插图、Logo。
* ⚙️ **Configs:** 启动器自定义主页代码 (`.xaml`/`.json`)、一些不便公开的半成品配置。
* 📜 **Docs:** 临时的说明文档、更新日志草稿。
* 🗑️ **Trash:** 一些或许有用，或许没用的历史遗留数据。
